# SogoApp — Landing & Legal Site

Static marketing + legal site for **SogoApp** (private messaging, calling & live streaming).

Pages:
- `/` — landing page
- `/delete-account/` — account & data deletion (Google Play requirement)
- `/privacy-policy/`
- `/terms-of-service/`
- `/support/`

Plain HTML/CSS — no build step. Hosted on GitHub Pages.
To use the custom domain `sogoappusa.com`, add it under repo **Settings → Pages → Custom domain** (a `CNAME` file will be created) and point DNS to GitHub Pages.
